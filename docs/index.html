BluTV Party Extension Installation Guide

Quick Install Steps:

1. Download the Extension

Download BluTV Party Extension

2. Extract the ZIP file

After downloading, extract the ZIP file to a folder on your computer.

3. Install in Chrome

  1. Open Chrome and go to chrome://extensions
  2. Enable "Developer mode" in the top right corner
  3. Click "Load unpacked"
  4. Select the folder where you extracted the extension files

How to Use:

  1. Go to any video on BluTV
  2. Click the extension icon in Chrome
  3. Create a room or join an existing one
  4. Share the room ID with friends
  5. Enjoy watching together!

Troubleshooting:

  • Make sure everyone is on the same video page
  • If the extension stops working, try clicking the extension icon and rejoining the room
  • If problems persist, try removing and reinstalling the extension
